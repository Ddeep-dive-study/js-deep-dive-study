## 암묵적 타입 변환

### 그게 뭔데
- 자바스크립트 엔진이 암묵적으로 데이터 타입을 변환함
```javascript
'10' + 2 // '102' <string>
5 * '10' // 50 <int>
if (1) {} // true <boolean>
```

- 템플릿 리터럴에서도 타입 변환 이루어짐
```javascript
`1+1 = ${1+1}` // 1+1 = 2
```

### 어디서 많이 쓸까
- 경험상 보통 숫자 <-> 스트링 변환에 많이 쓰는듯
```javascript
// 숫자 -> 스트링
NaN + '' // 'NaN'

//
```
